### Project Overview
This project is based on Java Spring MVC and mysql database。I used Bootstrap for front end。
### How to use
```
$ git clone https://github.com/withstars/Score-System

$ cd Score-System

$ mvn clean compile

$ mvn clean package

$ mvn clean install

$ mvn jetty:run

http://localhost:9000
```
